# 使用區段資格事件

區段資格活動可讓您的歷程聆聽 Adobe Experience Platform 區段中設定檔的入口和出口，讓個人得以在歷程中進入或前進。

範例使用案例可能是：

* 當行動服務客戶超過其資料限制時，則會觸發歷程
* 客戶到達新的忠誠度區段或退出忠誠度區段時

此影片簡要介紹如何設定區段，以及如何在歷程中設定區段資格事件。
