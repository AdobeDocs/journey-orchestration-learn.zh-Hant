# 跳到另一段歷程

瞭解如何將個人從一個歷程推向另一個歷程。

「跳轉」操作活動可讓您：

* 將複雜歷程分割為數個歷程，以簡化極為複雜歷程的設計
* 依據常見且可重複使用的歷程模式來建立歷程
